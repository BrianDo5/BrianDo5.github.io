Project 2: Futoshiki Game

Project Overview

The goal of this project was to create and develop a Java-based Futoshiki puzzle game. A Japanese logic game called futoshiki combines Sudoku with inequality restrictions. The object of the game is to complete a grid with numbers while adhering to the Sudoku rules and ensuring that the specified inequality signs (greater than or less than) between certain cells are satisfied.

Project 3: Word Split

Project Overview

The goal of this project was to create a Python software that could shrewdly identify words or phrases from lengthy, uninterrupted sequences of text, such URLs or file paths, based on linguistic norms and patterns.The application attempted to make such strings easier to read and to make text analysis easier.
